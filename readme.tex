1. Python files (Core Algorithms)
- lp.py
  -Implements the **LP baseline** method for differentially private query evaluation.

- nt.py
  - Implements the **Naive Truncation (NT) baseline** method.

-  R2T.py
  - Implements the **R2T algorithm** (instance-optimal truncation selection based approach).

2. Jupyter Notebooks
- PostgreSQL.ipynb
  - used to set up the PostgreSQL database:
  - create tables
  - insert data into tables

- experiments.ipynb
  - main experiment notebook:
  - runs **LP / NT / R2T** on aggregation queries: **SUM, COUNT, AVG, MIN, MAX**
  - collects algorithm outputs
  - computes error metrics
  - plots result figures for comparison

3. Datasets
- tpch-dbgen
  - Folder containing TPC-H dataset resources used for loading data into PostgreSQL.

- TPC-H data files (used for insertion into PostgreSQL):
  - customer.tbl
  - lineitem.tbl
  - nation.tbl
  - part.tbl
  - partsupp.tbl
  - orders.tbl
  - region.tbl
  - supplier.tbl
